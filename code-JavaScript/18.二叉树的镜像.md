## 题目描述

操作给定的二叉树，将其变换为源二叉树的镜像。

## 解题思路
树 -> 递归

对左子树做镜像+对右子树做镜像

```javascript
function Mirror(root)
{
    // write code here
    if(root === null){
        return null;
    }
    let temp;
    temp = root.right;
    root.right = root.left;
    root.left = temp;
    Mirror(root.left);
    Mirror(root.right);
}
```
