## 题目描述

统计一个数字在排序数组中出现的次数。

## 解题思路

此题的关键词： **排序数组**

所以，如果一个数字重复出现，必定是连续出现的。

可以暴力解法，也可以优雅一点：在js中有一个`indexOf()`可以找到第一次出现指定元素的索引（一般用于String，数组也能使用）

## Code


```javascript
function GetNumberOfK(data, k)
{
    // write code here
    var i = data.indexOf(k);
    if(i === -1) return 0;
    var count = 0;
    while(data[i] === k){
        count++;
        i++;
    }
    return count;
}
```
